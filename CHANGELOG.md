# Change Log

All notable changes to the "gigachad" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.5] - 2025-09-14

### Changed
- Remover funcionalidade de favoritos: não é mais possível marcar scripts como favoritos, nem acessar comandos ou menus relacionados.
- Alterar StatusBar para exibir apenas informações do histórico recente de execuções.
- Impedir duplicatas no histórico de scripts: ao executar novamente um script, apenas a data e as informações são atualizadas.
- Simplificar código com remoção de métodos, comandos e opções de favoritos.
